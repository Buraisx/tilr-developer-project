Contacts application for adding and removing contacts.
Inspiration from my phone's phone app.

Running on `Rails 5.1.7`

# REQUIRED 
- Ruby
- SQlite3
  - macs: `brew install sqlite3`

# SETUP
- `rails db:setup`
- `bundle install`

# STARTING UP
- `rails s`
- go to `localhost:3000`

# TESTING
- `rails test`
